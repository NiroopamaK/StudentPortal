# StudentPortal
This is simple Angular and SpringBoot project with basic CRUD operations and file handling. This is a portal for module configuration and adding student records for each  module to store student data in an efficient way. 

The application follows three tier architecture

- Frontend : Angular
- Backend : Spring Boot
- Database : MongoDB

-----------------------------------------------------------------------------

Prerequisites

- Java JDK 17 or a compatible version (JDK 17 has been used )
- Apache Maven (mvn 4.0 has been used)
- Node.js V20 (V16+ is recommended)
- Angular CLI
- MongoDB (Local installation)

----------------------------------------------------------------------------

Start the project

MongoDB - on port 27017
- mongod

Spring Boot Backend - on port 8080
- mvn clean install
- mvn spring-boot:run

Angular Frondend - on port 4200
- npm install
- ng serve
